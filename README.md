# Tentang Mintory
Mintory adalah Sistem Informasi Pengelolaan Sarana dan Prasarana yang dirancang untuk membantu instansi dalam melakukan pendataan barang, pengajuan pengadaan, serta pengelolaan peminjaman dan perbaikan barang secara terkomputerisasi.
Mintory menyediakan antarmuka yang sederhana, mudah digunakan, serta dilengkapi dengan pembagian hak akses antara Admin dan User.
#Hak Akses Pengguna
1. Admin

Admin memiliki kontrol penuh terhadap sistem:

Mengelola data barang

Mengelola data kategori & lokasi

Menerima/menolak pengajuan peminjaman

Menerima/menolak pengajuan perbaikan barang

Mengelola pengadaan barang

Melihat laporan riwayat barang

2. User

User memiliki hak untuk:

Mengajukan peminjaman barang

Mengajukan perbaikan barang

Melihat status pengajuan

Melihat daftar barang
