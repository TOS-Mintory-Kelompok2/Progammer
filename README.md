# Tentang Mintory
Mintory adalah Sistem Informasi Pengelolaan Sarana dan Prasarana yang dirancang untuk membantu instansi dalam melakukan pendataan barang, pengajuan pengadaan, serta pengelolaan peminjaman dan perbaikan barang secara terkomputerisasi.
Mintory menyediakan antarmuka yang sederhana, mudah digunakan, serta dilengkapi dengan pembagian hak akses antara Admin dan User.

# Tujuan Pengembangan
1. Mempermudah proses peminjaman barang.
2. Menyediakan mekanisme pelaporan kerusakan yang efektif.
3. Mempermudah pemantauan dan pengelolaan sarana prasarana.

# Hak Akses Pengguna
1. Admin
Admin memiliki kontrol penuh terhadap sistem:
- Mengelola data barang
- Mengelola data kategori & lokasi
- Menerima/menolak pengajuan peminjaman
- Menerima/menolak pengajuan perbaikan barang
- Mengelola pengadaan barang
- Melihat laporan riwayat barang

2. User
User memiliki hak untuk:
- Mengajukan peminjaman barang
- Mengajukan perbaikan barang
- Melihat status pengajuan
- Melihat daftar barang

# Fitur Mintory

1. Manajemen Pengguna
- Login & logout
- Role admin & user
- Admin dapat mengelola data pengguna

2. Manajemen Data Barang
- Tambah, edit, hapus, dan lihat data barang
- Melihat detail: nama, kategori, kondisi, jumlah, status

3. Pengajuan Peminjaman Barang
- User mengajukan peminjaman
- Sistem menyimpan pengajuan
- Admin menyetujui atau menolak pengajuan
- User melihat status peminjaman

4. Pengajuan Perbaikan Barang
- User mengajukan laporan kerusakan
- Sistem mencatat laporan
- Admin memproses dan memantau perbaikan

5. Pengadaan Barang
- Admin mengelola data pengadaan
- Riwayat pengadaan tersimpan otomatis

6. Laporan dan Monitoring
- Rekap peminjaman, perbaikan, pengadaan
- Menampilkan status dan histori sistem
